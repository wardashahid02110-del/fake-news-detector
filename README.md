# 📰 Fake News Detector

A machine learning project to detect fake news articles using NLP techniques.

## 📊 Dataset

Download the dataset from Kaggle:
👉 [Fake and Real News Dataset](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset)

After downloading, place these files in the project folder:
- `Fake.csv` — 23,481 fake news articles
- `True.csv` — 21,417 real news articles

## 📁 Project Structure

```
fake-news-detector/
│
├── preprocess.py       # Data loading and cleaning
├── requirements.txt    # Python dependencies
├── .gitignore          # Files to ignore
└── README.md           # Project info
```

## ⚙️ Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/fake-news-detector.git
cd fake-news-detector

# 2. Install dependencies
pip install -r requirements.txt

# 3. Download dataset from Kaggle and place CSVs here

# 4. Run preprocessing
python preprocess.py
```
